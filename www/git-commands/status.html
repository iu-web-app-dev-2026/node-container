git status

The git status command is used to display the current state of the repository

Example: git status

Output:

    On branch feature/add-status
    Your branch is up to date with 'origin/feature/add-status'.

    Untracked files:
    (use "git add file..." to include in what will be committed)
            www/git-commands/status.html
    nothing added to commit but untracked files present (use "git add" to track)
