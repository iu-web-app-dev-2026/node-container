Server Information

Port: 8080

Package: http-server

Directory: ./www

Getting Started

To start the server, run:

  • npm start - Starts server and opens browser
  • npm run dev - Starts server only

Add your HTML, CSS, and JavaScript files to the www directory to serve them.
